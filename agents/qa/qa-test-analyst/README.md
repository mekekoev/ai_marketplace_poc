# qa-test-analyst

Coordinate placeholder functional QA smoke analysis to confirm QA plugin skills and agents import correctly from the marketplace.

---

## Install

npx skills add git@git.epam.com:epmt-rtqa/qe-agent-skills.git --agent qa-test-analyst

---

## What This Agent Does

This placeholder agent coordinates a small functional QA marketplace smoke check. It routes the request to the packaged placeholder skills, reports what was exercised, and avoids production-facing actions.

---

## When It Triggers

- "Use qa-test-analyst to validate marketplace import."
- "Smoke test the functional QA plugin agent."
