# perf-assempble-test

Create a placeholder performance assembly test skill for verifying skills_registry-to-marketplace plugin assembly.

---

## Install

npx skills add git@git.epam.com:epmt-rtqa/qe-agent-skills.git --skill perf-assempble-test

---

## What This Skill Changes

| Without skill | With skill |
|---|---|
| Marketplace assembly smoke tests cannot prove a newly added performance skill is discovered. | A small performance testing placeholder can be installed and assembled. |
| Script discovery cannot be checked for this placeholder. | Two deterministic scripts are available for smoke checks. |

---

## When It Triggers

- "Use perf-assempble-test to smoke test marketplace assembly."
- "Check that the performance testing placeholder skill is assembled from skills_registry."
