# security-owasp-smoke

Create compact OWASP-style security smoke checklists for marketplace import validation and security plugin testing.

---

## Install

npx skills add git@git.epam.com:epmt-rtqa/qe-agent-skills.git --skill security-owasp-smoke

---

## What This Skill Changes

| Without skill | With skill |
|---|---|
| Marketplace import smoke tests rely on a missing security testing skill. | A small security testing placeholder can be installed and invoked. |
| Script discovery cannot be checked for this discipline. | Two deterministic scripts are available for smoke checks. |

---

## When It Triggers

- "Use security-owasp-smoke to smoke test marketplace import."
- "Check the security testing plugin placeholder skill."
