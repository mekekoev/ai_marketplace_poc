# perf-k6-smoke-check

Create small k6-oriented performance smoke checks for marketplace validation and performance plugin discovery tests.

---

## Install

npx skills add git@git.epam.com:epmt-rtqa/qe-agent-skills.git --skill perf-k6-smoke-check

---

## What This Skill Changes

| Without skill | With skill |
|---|---|
| Marketplace import smoke tests rely on a missing performance testing skill. | A small performance testing placeholder can be installed and invoked. |
| Script discovery cannot be checked for this discipline. | Two deterministic scripts are available for smoke checks. |

---

## When It Triggers

- "Use perf-k6-smoke-check to smoke test marketplace import."
- "Check the performance testing plugin placeholder skill."
