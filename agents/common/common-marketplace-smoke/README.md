# common-marketplace-smoke

Coordinate a lightweight common-plugin marketplace smoke test that verifies packaged skills and agents can be discovered.

---

## Install

npx skills add git@git.epam.com:epmt-rtqa/qe-agent-skills.git --agent common-marketplace-smoke

---

## What This Agent Does

This placeholder agent coordinates a small common QE marketplace smoke check. It routes the request to the packaged placeholder skills, reports what was exercised, and avoids production-facing actions.

---

## When It Triggers

- "Use common-marketplace-smoke to validate marketplace import."
- "Smoke test the common QE plugin agent."
