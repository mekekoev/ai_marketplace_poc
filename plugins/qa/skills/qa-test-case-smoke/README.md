# qa-test-case-smoke

Create lightweight functional test case smoke checklists for marketplace import validation and QA plugin discovery tests.

---

## Install

npx skills add git@git.epam.com:epmt-rtqa/qe-agent-skills.git --skill qa-test-case-smoke

---

## What This Skill Changes

| Without skill | With skill |
|---|---|
| Marketplace import smoke tests rely on a missing functional QA skill. | A small functional QA placeholder can be installed and invoked. |
| Script discovery cannot be checked for this discipline. | Two deterministic scripts are available for smoke checks. |

---

## When It Triggers

- "Use qa-test-case-smoke to smoke test marketplace import."
- "Check the functional QA plugin placeholder skill."
