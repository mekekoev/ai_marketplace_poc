# Domain — Platform-Specific Testing

Want to contribute a skill? See the [contribution guide](../../docs/CONTRIBUTING.md).

This folder is flat — all domain skills sit directly here, no sub-folders per platform.
Skill names must include a platform prefix:

| Platform | Prefix | Example |
|---|---|---|
| SAP | `sap-*` | `sap-test-case-generator` |
| Salesforce | `salesforce-*` | `salesforce-flow-validator` |
| Game | `game-*` | `game-regression-checklist` |
| Blockchain | `blockchain-*` | `blockchain-smart-contract-tester` |
| Adobe AEM | `aem-*` | `aem-component-tester` |
