# aqa-test-automation-planner

Coordinate placeholder automation planning checks to verify AQA plugin skills and agents are visible after marketplace import.

---

## Install

npx skills add git@git.epam.com:epmt-rtqa/qe-agent-skills.git --agent aqa-test-automation-planner

---

## What This Agent Does

This placeholder agent coordinates a small automated testing marketplace smoke check. It routes the request to the packaged placeholder skills, reports what was exercised, and avoids production-facing actions.

---

## When It Triggers

- "Use aqa-test-automation-planner to validate marketplace import."
- "Smoke test the automated testing plugin agent."
