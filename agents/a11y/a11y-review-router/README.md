# a11y-review-router

Route lightweight accessibility smoke review requests to validate a11y plugin import and placeholder skill availability.

---

## Install

npx skills add git@git.epam.com:epmt-rtqa/qe-agent-skills.git --agent a11y-review-router

---

## What This Agent Does

This placeholder agent coordinates a small accessibility testing marketplace smoke check. It routes the request to the packaged placeholder skills, reports what was exercised, and avoids production-facing actions.

---

## When It Triggers

- "Use a11y-review-router to validate marketplace import."
- "Smoke test the accessibility testing plugin agent."
