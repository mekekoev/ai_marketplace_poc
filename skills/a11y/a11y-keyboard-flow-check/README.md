# a11y-keyboard-flow-check

Draft small keyboard navigation smoke checks for validating accessibility marketplace installation and skill discovery.

---

## Install

npx skills add git@git.epam.com:epmt-rtqa/qe-agent-skills.git --skill a11y-keyboard-flow-check

---

## What This Skill Changes

| Without skill | With skill |
|---|---|
| Marketplace import smoke tests rely on a missing accessibility testing skill. | A small accessibility testing placeholder can be installed and invoked. |
| Script discovery cannot be checked for this discipline. | Two deterministic scripts are available for smoke checks. |

---

## When It Triggers

- "Use a11y-keyboard-flow-check to smoke test marketplace import."
- "Check the accessibility testing plugin placeholder skill."
