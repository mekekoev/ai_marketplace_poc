# perf-load-test-planner

Coordinate lightweight performance smoke planning to confirm performance plugin import and placeholder skill availability.

---

## Install

npx skills add git@git.epam.com:epmt-rtqa/qe-agent-skills.git --agent perf-load-test-planner

---

## What This Agent Does

This placeholder agent coordinates a small performance testing marketplace smoke check. It routes the request to the packaged placeholder skills, reports what was exercised, and avoids production-facing actions.
