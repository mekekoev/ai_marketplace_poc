# security-review-planner

Coordinate small security review smoke checks to validate security plugin installation and placeholder skill discovery.

---

## Install

npx skills add git@git.epam.com:epmt-rtqa/qe-agent-skills.git --agent security-review-planner

---

## What This Agent Does

This placeholder agent coordinates a small security testing marketplace smoke check. It routes the request to the packaged placeholder skills, reports what was exercised, and avoids production-facing actions.

---

## When It Triggers

- "Use security-review-planner to validate marketplace import."
- "Smoke test the security testing plugin agent."
