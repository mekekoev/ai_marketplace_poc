# aqa-api-test-smoke

Prepare lightweight API automation smoke scenarios for validating AQA marketplace installation and skill import behavior.

---

## Install

npx skills add git@git.epam.com:epmt-rtqa/qe-agent-skills.git --skill aqa-api-test-smoke

---

## What This Skill Changes

| Without skill | With skill |
|---|---|
| Marketplace import smoke tests rely on a missing automated testing skill. | A small automated testing placeholder can be installed and invoked. |
| Script discovery cannot be checked for this discipline. | Two deterministic scripts are available for smoke checks. |

---

## When It Triggers

- "Use aqa-api-test-smoke to smoke test marketplace import."
- "Check the automated testing plugin placeholder skill."
